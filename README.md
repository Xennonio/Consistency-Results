# Notes on Cofinality
This repository contains the entire LaTeX project of my personal notes on Consistency Results, based on Kunen's book - An Introduction to Independence Proofs.

## Pdf File
The compiled version of these notes is available [here](https://raw.githubusercontent.com/Xennonio/Consistency-Results/main/Consistency_Results.pdf).

## About the Notes
These notes are merely a comprehensive and filtered summary of Kunen's Book - Set Theory: An Introduction to Independence Proofs, along with some of my own intuitions that may be useful to someone, and some additional comments on things I've found elsewhere.
